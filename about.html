Installing Vanta

1. Self-install
Vanta self-install
If it doesn't work try this:
./vanta.exe self-install
2. Implementing Vanta
Vanta init
If it doesn't work then try this:
./Vanta.exe init
If all above don't work try this:
cd path\to\vanta
